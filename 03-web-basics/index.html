ブラウザの裏側をのぞいてみよう

URL を打ってから画面が出るまでの流れと、HTTP ステータスコードを体験できます。

① URL を打ってから画面が出るまで

1
URL を解釈
ブラウザが「どこに・どうやって」接続するか読み取る
2
DNS に問い合わせ
example.com の IP アドレスは？ → 93.184.216.34
3
HTTP リクエストを送る
その IP に「このページください」とお願いする
4
HTTP レスポンスが返る
200 OK ＋ HTML 本文が返ってくる
5
DOM を組み立てる
HTML を読んで木構造のデータにする
6
画面に描画（レンダリング）
CSS を当てて JS を動かし、ピクセルにする

「アクセス」を押すと、上から順に光っていきます。DevTools の Network タブを開いた状態で押すと、実際の通信と見比べられます。

② HTTP ステータスコードを体験

ボタンを押すと、サーバーからの「返事」が表示されます。

上のボタンを押してみてください。

本物のステータスコードを見たいときは、ブラウザで https://http.cat/404 のように開くと猫の画像で返ってきます。
